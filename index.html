Welcome to 3D Zoning App Prototype
Created by Esri Indonesia as demo for Dinas Cipta Karya, Tata Ruang dan Pertanahan DKI Jakarta. This app is meant to demonstrate 3D GIS capabilities in city planning. Developed with ArcGIS API for Javascript 4.5.
Disclaimer: Data and information contained in this app is for demo purpose only and can not be used as a reference.
